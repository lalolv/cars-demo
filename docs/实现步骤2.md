# 汽车展示台 - 实现步骤 2（阶段八：大屏 + 灯光）

## 目标

在现有主场景基础上完成展厅布局升级，加入后场主大屏与展车造型灯组，并提供可验收的检查清单。

## 8.1 后场大屏搭建

目标：在主车后方加入品牌主屏，支持图片轮播（PPT 风格）。

1. 在 `Main` 下新增 `BackScreenRoot (Node3D)`。
2. 新增 `MainScreen (MeshInstance3D)`，使用 `QuadMesh`（推荐比例 16:9）。
3. 新增 `ScreenSlideshow (Node)` 并挂载 `scripts/screen_slideshow.gd`。
4. 在 `slides` 中配置图片数组，设置 `seconds_per_slide` 和 `fade_in_duration`。

## 8.2 展车造型灯组

目标：提升车身体积感和线条表现，保证移动端开销可控。

1. 新增 `LightingRig (Node3D)`。
2. 新增 `KeyLightLeft`、`KeyLightRight`、`RimLight` 三盏补光灯。
3. 仅关键灯开启阴影，其他灯关闭阴影。
4. 调整灯色和强度，确保主视觉仍聚焦车辆本体。

## 8.3 验收清单

- [ ] 大屏轮播可自动循环，切换无突兀闪烁。
- [ ] 大屏亮度不压制车身高光。
- [ ] 车辆在前 3/4 角度有清晰明暗层次。
- [ ] 移动端运行无明显帧率抖动。
- [ ] 切换车型后灯光表现稳定。

## 第二阶段工作（进行中）

围绕展厅布局方案中的阶段 B 执行如下内容：

1. 增加 `BrandWall` 与 `SpecWall` 展厅背景体块。
2. 调整舞台地面材质反射与粗糙度，提升质感。
3. 增加 UI 控件：大屏开关与轮播速度调节。
